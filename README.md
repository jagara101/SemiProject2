- 서울정보소통광장(https://opengov.seoul.go.kr)에 게시되어있는 "관광숙박업 등록 현황" 2022년 자료를 선정하여 프로젝트 진행.

- 데이터 전처리 후 총 6가지 방법을 이용하여 시각화 진행(Folium, 도수분포표, 히스토그램, 막대그래프, 파이차트, SVG지도시각화)
